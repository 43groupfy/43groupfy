# List 란?

- 순서가 있는 데이터들의 집합
- 불연속적인 메모리 공간에 데이터들이 연관되며 포인터를 통해 각 데이터들이 연결
- 리스트는 동적으로 크기가 결정
- 단순  배열에 비해 데이터의 삽입, 삭제가 용이하고 메모리의 재사용성이 높음

# List 종류

1. ArrayList
2. LinkedList
