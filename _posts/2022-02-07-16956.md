# [C++] BOJ 16956 늑대와 양

## 문제

[BOJ 16956 늑대와 양](https://www.acmicpc.net/problem/16956)

## 풀이

늑대가 있는 곳 주위에 울타리를 치면 된다. 만약 울타리를 쳐야하는 그 위치에 양이 있다면 0을 출력하고 바로 끝낸다.

## 코드

```c++
#include <cstdio>
using namespace std;
int dy[] = {0, 1, 0, -1}, dx[] = {-1, 0, 1, 0};

int n, m;
char arr[500][500];

int main() {
    scanf("%d %d", &n, &m);
    for(int i = 0; i < n; i++) {
        for(int j = 0; j < m; j++) {
            scanf(" %c", &arr[i][j]);
        }
    }

    for(int i = 0; i < n; i++) {
        for(int j = 0; j < m; j++) {
            if (arr[i][j] == 'W') {
                for(int k = 0; k < 4; k++) {
                    int x = i + dx[k], y = j + dy[k];
                    if (x < 0 or y < 0 or x >= n or y >= m) continue;
                    if (arr[x][y] == 'W') continue;
                    if (arr[x][y] == 'S') {
                        printf("0");
                        return 0;
                    }
                    arr[x][y] = 'D';
                }
            }
        }
    }
    printf("1\n");
    for(int i = 0; i < n; i++) {
        for(int j = 0; j < m; j++) {
            printf("%c", arr[i][j]);
        }
        printf("\n");
    }
}
```
