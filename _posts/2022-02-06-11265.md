# [C++] BOJ 11265 끝나지 않는 파티

## 문제

[BOJ 11265 끝나지 않는 파티](https://www.acmicpc.net/problem/11265)

## 풀이

폴로이드-와샬의 기본 유형이다. 경유해서 가는 것이 더 빠르면 `adj[i][j]`의 값을 갱신시켜준다.

## 코드

```c++
#include <cstdio>
#include <algorithm>
using namespace std;

int n, m;
int adj[500][500];

void go() {
    for(int k = 0; k < n; k++) {
        for(int i = 0; i < n; i++) {
            for(int j = 0; j < n; j++) {
                adj[i][j] = min(adj[i][j], adj[i][k] + adj[k][j]);
            }
        }
    }
}

int main() {
    scanf("%d %d", &n, &m);
    for(int i = 0; i < n; i++) {
        for(int j = 0; j < n; j++) {
            scanf("%d", &adj[i][j]);
        }
    }
    go();
    int s, e, t;
    for(int i = 0; i < m; i++) {
        scanf("%d %d %d", &s, &e, &t);
        s--; e--;
        if (adj[s][e] <= t) printf("Enjoy other party\n");
        else printf("Stay here\n");
    }
}
```
